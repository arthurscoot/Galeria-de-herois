# Imersão Dev Alura - Galeria de Heróis

Este projeto é uma galeria de heróis interativa, desenvolvida durante a 10ª edição da Imersão Dev da Alura.

##  Propósito do Projeto

A "Galeria de Heróis" foi criada para ser uma ferramenta útil tanto para jogadores novatos quanto para veteranos de jogos de heróis baseados em equipes. O site serve como um guia rápido e visual, fornecendo informações essenciais sobre cada personagem, ajudando os jogadores a:

- Conhecer os diferentes heróis e suas funções (Tanque, Dano e Suporte).
- Entender a história e o papel de cada personagem no universo do jogo.
- Visualizar o design e as habilidades dos heróis através de imagens e vídeos.

##  O que o site faz?

O site exibe uma coleção de cartões, cada um representando um herói. Ao interagir com a galeria, o usuário pode ver:

- O nome e o título do herói.
- A sua função principal no jogo.
- Uma breve descrição sobre o personagem.
- Imagens e links para vídeos de apresentação.
